🛡️ Prowler Helm Chart Repository

Kubernetes Cloud Security AWS Azure GCP

Welcome to the official Helm chart repository for Prowler - an Open Cloud Security tool for AWS, Azure, GCP and Kubernetes.

📦 Installation

Add this Helm repository:

helm repo add prowler https://guilhermealegre.github.io/prowler-helm-chart/
helm repo update

Search for available charts:

helm search repo prowler

Install the chart:

helm install my-prowler prowler/prowler
