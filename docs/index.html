Prowler Helm Chart Repository

Welcome to the Helm Chart repository for Prowler.

Usage

To use this chart repository, add it to your Helm repositories:

helm repo add prowler-helm-chart https://guilhermealegre.github.io/prowler-helm-chart/
helm repo update

Then, you can install the chart:

helm install my-prowler prowler-helm-chart/prowler
